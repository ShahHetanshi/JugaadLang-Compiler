# 🛠️ JugaadLang Compiler

JugaadLang Compiler is a custom-built compiler for **JugaadLang**, a lightweight programming language designed to demonstrate core compiler design concepts such as lexical analysis, parsing, semantic analysis, and code generation.

This project was built to understand the internal working of compilers and to implement fundamental language processing techniques from scratch.


## 🚀 Project Overview

The compiler processes JugaadLang source code through the following stages:

1. **Lexical Analysis** – Converts source code into tokens  
2. **Syntax Analysis (Parsing)** – Builds an Abstract Syntax Tree (AST)  
3. **Semantic Analysis** – Performs validation like type checking and scope resolution  
4. **Code Generation** – Produces executable output / intermediate representation  


## ⚙️ Features

- Custom programming language syntax
- Tokenizer (Lexer)
- Parser with grammar rules
- Basic semantic checks
- Error handling with meaningful messages
- Sample test programs
